Space Bar Labs APT Repository

About apt.spacebarlabs.com

This is a Debian APT repository providing curated packages and meta-packages to make it easy to install preferred tools on Debian-based systems. The repository includes custom packages for development tools, utilities, and applications maintained by Space Bar Labs.
